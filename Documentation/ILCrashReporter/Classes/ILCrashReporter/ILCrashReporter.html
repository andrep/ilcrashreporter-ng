ILCrashReporter

Crash reporter task
Superclass: NSObject


The ILCrashReporter class enables you to launch a background task that will monitor the launching application. If the application unexpectedly terminates the crash reporter will display a notification and allow the user to provide a description of the nature of the crash. If the user desides to complete the crash report an email is sent containing the users description and the crashlog generated by the operating system.
